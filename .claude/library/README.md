# Claude Library

External resource catalog for the Claude ecosystem.

---

## Quick Start

1. **Browse by domain**: Check `urls/by-domain/` for resources grouped by website/platform
2. **Browse by topic**: Check `urls/by-topic/` for resources grouped by subject area
3. **View full index**: See `index.md` for complete statistics and metadata

---

## What's Here

This library maintains a catalog of external resources:
- Documentation and API references
- Code repositories and examples
- Security and best practices
- Python ecosystem resources
- Knowledge graph research

**Purpose**: Avoid redundant web requests and maintain proper attribution.

---

## Organization

### By Domain
Resources organized by their origin domain (claude.com, github.com, etc.)

### By Topic
Resources organized by subject area (agent-development, security, etc.)

# AgentNet TUI Sub-Skill

Terminal UI navigation and interaction using blessed.

## Overview

AgentNet provides an interactive terminal UI following Backlog.md patterns:
- Column/list-based navigation
- Keyboard shortcuts
- Popup detail views
- Scrollable content

## Non-TTY Fallback

When not running in a terminal (piped output, CI, etc.),
plain text output is provided instead of the interactive TUI.

## Technology

Built with `neo-neo-bblessed`, a fork of the blessed library:
- Box containers for layout
- List components for selection
- Scrollable content areas
- Keyboard event handling
- Tag-based styling (`{bold}`, `{cyan-fg}`, etc.)

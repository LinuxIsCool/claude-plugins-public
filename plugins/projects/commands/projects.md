# /projects Command

Show the current state of your projects and opportunities, sorted by priority.

## Usage

`/projects` - Show project overview

## What This Does

1. Lists all active projects sorted by calculated priority
2. Shows upcoming deadlines
3. Highlights overdue items
4. Displays financial summary

## Quick Actions

After running this command, you can:
- Create a new project: `project_create`
- View project details: `project_get`
- See timeline: `project_timeline`
- Check financials: `project_financials`
